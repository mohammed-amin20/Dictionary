<h1>Dictionary</h1>

## 📖 About the App

Dictionary is a simple and clean Android app that allows users to search 
for any English word and get its full definition instantly. It displays 
the word's meaning, phonetic transcription, pronunciation audio, and 
examples grouped by part of speech — all in a smooth and easy-to-use interface.

Built with Kotlin and Jetpack Compose.
